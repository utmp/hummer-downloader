## Vue + Vite + Tailwindcss 
### Reusable project for frontend devs (***pre-configured***).
### Clone 
``` 
git clone https://utmp/vue-vite-tw.git
```
### Intall
```
npm install
```
### Run 
```
npm run dev
```
### Package
```
npm run package
```
